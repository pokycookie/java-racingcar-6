# 기능 구현 목록

- [x] 사용자 입력을 통해 경주 할 자동차를 생성할 수 있어야 함
- [x] 각 차수별로 랜덤한 값에 따라 전진여부를 결정하고, 자신의 이동을 기억하고 있어야 함
- [x] 사용자 입력, 차수 진행, 결과 발표와 같은 게임의 흐름을 통제할 수 있어야 함

### 반복 횟수 입력

- [x] 숫자 형태의 입력이어야 함
- [ ] 음수가 아닌 정수 형태여야 함 (0 이상의 정수, 0을 허용할지 말지 고민중...)
- [ ] int 형 범위를 벗어나는 정수는 불가
